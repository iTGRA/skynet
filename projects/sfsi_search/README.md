# sfsi.ru semantic search

Проект для семантического поиска по сайту sfsi.ru.

Цель: собрать страницы сайта, преобразовать их в векторное представление и хранить индексы в Qdrant. Для анализа и формирования ответов используется локальная LLM через Ollama.

Основные задачи:
- загрузка и разбор страниц sfsi.ru;
- подготовка текста для индексации;
- сохранение векторных представлений в Qdrant;
- поиск релевантных материалов по смыслу;
- работа без внешних облачных моделей, с локальным LLM.
